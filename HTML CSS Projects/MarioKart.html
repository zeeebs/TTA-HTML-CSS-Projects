Mario Kart - 1992 to present day 2023

Released in 1992 for the Super Nintendo Enterntainement System, Super Mario Kart and was the very first Mario Kart game ever released.
Since its conception, there have been many iterations of the game,
spanning multiple consoles. These iterations include changes to: characters,
vehicles, game play, controls, and track design, and each game has used
its predecesors to propell it to the next level.
